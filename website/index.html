Features

Built for the way meetings actually sound.

Two audio streams, separately labelled. Live captions while you talk. Structured notes when you stop.

Dual audio capture

Microphone via Web Audio API, system audio via AudioTee. Both streams are downsampled to 16 kHz PCM and transcribed in parallel — your voice and theirs never blend.

Real-time transcription

Words land on screen as they're spoken, via Deepgram's nova-2 streaming WebSocket — or run local Whisper for fully offline.

Markdown on disk

Each meeting is a folder containing transcript.md, notes.md, and metadata.json. Open them in any editor.

AI-powered notes

On-demand summaries, decisions, and action items via any OpenAI chat model — pick from a dropdown — or a local Ollama model. Generation is always opt-in.

Smart organization

Auto-sorted by recency. Drag-and-drop sections, color tags, and full-text search across every transcript you've ever recorded.

Local weekly calendar

Plan upcoming meetings on a 7-day grid, then start a recording straight from the event with the title and tag pre-filled. Stored locally — no Google or Outlook accounts required.

Cloud or fully offline

Pair Deepgram + OpenAI for the lowest latency, or use Whisper + Ollama and never make a single network request.

How it works

From sound to structured notes.

The full lifecycle of one meeting, from the moment you hit record.

Step 01

Capture

Mic via Web Audio API, system audio via AudioTee. Streams are downsampled to 16 kHz PCM.

Step 02

Transcribe

Two parallel Deepgram WebSockets — or a local Whisper model running entirely on your machine.

Step 03

Display & save

Live captions render with speaker labels. On stop, Markdown is written to your storage folder.

Step 04

Generate

Hit the notes button. Your chosen OpenAI or Ollama model returns a summary, decisions, and action items.

Privacy

Your meetings, your machine.

No backend. No telemetry. No subscription. Plain Markdown files in a folder you choose.

No backend

No accounts, no servers

MINT has no servers. Network calls are made only to the providers you configure — and only while you're recording or generating notes.

No telemetry

The app doesn't phone home

No analytics, no event tracking, no transcripts uploaded anywhere you didn't explicitly point the app at.

Plain text

Yours forever

Meetings are stored as Markdown. If you stop using MINT tomorrow, every transcript still opens in any text editor on earth.

Install

Four commands. No app store.

MINT is distributed as source. You build it locally — which keeps macOS Gatekeeper out of the way.

1

Prerequisites

macOS on Apple Silicon, Node 20+, Git, and a free Deepgram API key.

2

Clone the repo

Pull the source from GitHub.

3

Install & build

Run npm install then npm run build:mac. Drag the resulting .dmg into Applications.

4

Add your keys

Open Settings inside the app and paste your Deepgram (and optionally OpenAI) keys. No .env.

Terminal 
# Clone, install, build a local DMG.
$git clone https://github.com/ChrisWoo0443/MINT.git
$cd MINT
$npm install
$npm run build:mac

# Drag dist/MINT-*.dmg into Applications.
# To update later:
$git pull && npm run build:mac
